Відповіді на контрольні питання (варіант 11)
1. Що таке індексатор?
Індексатор - це спеціальний член класу в C# (або перевантажений оператор [] у C++), який дозволяє звертатися до об'єкта як до масиву, використовуючи індекс у квадратних дужках.

2. Для чого застосовують індексатор?
Індексатори застосовують для:

Доступу до елементів колекції у класі

Реалізації альтернативного синтаксису доступу до даних

Інкапсуляції логіки доступу до елементів

3. Поясніть призначення ексессорів get та set.

get - визначає логіку отримання значення

set - визначає логіку присвоєння значення
У моєму коді для індексатора використовується лише get, оскільки завдання не вимагає зміни діагоналей.

4. Що таке властивість?
Властивість - це член класу, який забезпечує механізм для читання, запису або обчислення значення приватного поля. У C++ аналогом є методи get/set.

5. Для чого застосовують властивості?
Властивості застосовують для:

Інкапсуляції доступу до полів класу

Додавання логіки при отриманні/зміні значень

Створення обчислюваних значень

6. Що таке автоматична властивість? Поясніть її призначення
Автоматична властивість - це властивість, де компілятор автоматично створює приватне поле та реалізацію get/set. Використовується, коли не потрібна додаткова логіка.

7. Чи підлягають перевантаженню індексатори та властивості? Якщо так, то навести приклади.
Так, індексатори можна перевантажувати за параметрами. Наприклад:
public int this[int i] { ... }
public int this[string s] { ... }
Властивості перевантажувати не можна, оскільки вони не приймають параметрів.

8. Які типи індексів дозволяють застосовувати індексатори?
Індекси можуть бути будь-якого типу (цілі числа, рядки тощо). У моєму коді використано цілий тип для вибору діагоналі (0 або 1).
